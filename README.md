## Guia
Neste repositório temos três diretórios sendo cada um destes dedicados a uma área aplicável do SQL; sendo o primeiro dedicado aos fundamentos do SQL, o segundo voltado a questões de performance em instruções SQL e por fim o último que tem como principio utilizar o SQL para a análise de dados. Em cada diretório deixo indicado quais fontes estou usando para aquele tipo de estudos, porém acredito que seja valido deixar nesta página quais são os livros que estou utilizando em cada área de estudo SQL, estas são: 

- **Introdução à Linguagem SQL** Por Thomas Nield 
- **SQL Guia Prático: Um guia para o uso de SQL** Por Alice Zhao
- **SQL para Análise de Dados: Técnicas avançadas para transformar dados em insights** Por Cathy Tanimura
